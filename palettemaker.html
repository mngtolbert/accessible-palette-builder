Accessible Color Palette Builder

Built for instructional designers — verify WCAG contrast compliance before your colors go into Rise 360, Storyline, or Figma
How to use
Paste your hex codes below — one per line, with an optional name. Hit Build Palette to see each color with its WCAG contrast rating against white and black text. Use the verdict label to decide whether a color is safe for body text, large headings only, or decorative elements only.
WCAG quick reference
AAACR ≥ 7:1 · Safe for all text
AACR ≥ 4.5:1 · Safe for body text
FailsCR < 4.5:1 · Decorative use only
